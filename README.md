# Simple Harmonic Motion Simulation

This project simulates Simple Harmonic Motion (SHM) and damped oscillations using HTML, CSS, and JavaScript.

## Physics Model

Ideal SHM:
F = -kx  
x(t) = A cos(ωt)

Damped SHM:
x(t) = A e^(-γt) cos(ωt)

## Features
- Adjustable amplitude
- Adjustable damping
- Real-time animation using canvas

## Concepts Demonstrated
- Simple Harmonic Motion
- Damped oscillations
- Energy loss in real systems
